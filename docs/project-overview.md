# ART3MIS SOC AI Project Overview

ART3MIS SOC AI is a portfolio-grade cybersecurity and AI project.

The project simulates a SOC-style dashboard where logs are uploaded, analyzed, scored, summarized, and turned into security reports.

## Why This Project Matters

This project demonstrates practical skills in:

- Cybersecurity log analysis
- Python backend development
- REST APIs
- Frontend dashboard design
- AI-assisted incident reporting
- Technical documentation
