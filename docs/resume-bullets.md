# Resume Bullets

- Built ART3MIS SOC AI, a full-stack cybersecurity dashboard using SvelteKit and Python FastAPI to analyze logs and generate incident summaries.
- Implemented rule-based detection for failed logins, suspicious IP activity, admin/root targeting, and SQL injection-style patterns.
- Developed REST API endpoints for text analysis, log file uploads, and structured JSON threat reports.
- Designed a SOC-style frontend dashboard showing severity counts, suspicious events, and AI-assisted remediation guidance.
- Created technical documentation covering project architecture, threat scoring, and portfolio presentation.
